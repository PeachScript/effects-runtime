# Galacean Effects Media Plugin

## Usage

### Simple Import

``` ts
import { Player } from '@galacean/effects';
import '@galacean/effects-plugin-media';
```

## Development

### Getting Started

``` bash
# demo
pnpm --filter @galacean/effects-plugin-media dev
```

> Open in browser: http://localhost:8081/demo/
